Usages :


Each of these colors is defined with a base color class and an optional lighten or darken class or accent class as well as hex color code.

Background Color :


To apply a background color, just add the color name and accent/light/darkness as a class to the element. 
						<div class="red">red color displayed as a background color.</div>
Example:
red color displayed as a background color.
Text Color :


To apply a text color, just append -text to the color class like this:
						<div class="blue-text">blue color as a text color displayed.</div>
